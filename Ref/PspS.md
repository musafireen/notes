# c1
अथ  प्रथमो ऽध्यायः    
### 1.1
अथातः पशुपतेः पाशुपतं योग-विधिं व्याख्यास्यामः // 1।1 //  
Now, therefore, we shall explain the method of the Pāśupata Yoga of Paśupati.
### 1.2
भस्मना त्रिषवणं स्नायीत // 1।2 //  
One should bathe thrice daily with ashes.
### 1.3
भस्मनि शयीत // 1।3 //  
One should sleep on ashes.

अनुस्नानं // 1।4 //  
Post-bath purification.
### 1.5
निर्माल्यं // 1।5 //  
Discarding of worn-out flowers or offerings.
### 1.6
लिङ्ग-धारी // 1।6 //  
Wearing the Liṅga.
### 1.7
आयतन-वासी // 1।7 //  
Residing in a sacred place (temple, monastery, etc.).
### 1.8
हसित-गीत-नृत्त-दुन्दुं-कार-नमस्कार-जप्योपहारेणोपतिष्ठेत् // 1।8 //  
One should worship with laughter, song, dance, drum sounds, salutations, chanting, and offerings.
### 1.9
महादेवस्य दक्षिणामूर्तेः // 1।9 //  
(Meditating) on the Dakṣiṇāmūrti form of Mahādeva.
### 1.10
एक-वासाः // 1।10 //  
Wearing a single garment.
### 1.11
अवासा वा // 1।11 //  
Or remaining unclothed.
### 1.12
मूत्रपुरीषं नावेक्षेत् // 1।12 //  
One should not look at urine or feces.
### 1.13
स्त्री-शूद्रं नाभिभाषेत् // 1।13 //  
One should not converse with women or Śūdras.
### 1.14
यद्यवेक्षेत्यद्यभिभाषेत् // 1।14 //  
If one does look at or converse (with them)...
### 1.15
उपस्पृश्य // 1।15 //  
...one should perform purification.
### 1.16
प्राणायामं कृत्वा // 1।16 //  
Having practiced Prāṇāyāma.
### 1.17
[[TA#10.20]], [[TA#10.1#5]]
रौद्रीं गायत्रीं बहु-रूपीं वा जपेत् // 1।17 //  
One should chant the Raudrī Gāyatrī or one with multiple forms.
### 1.18
अकलुष-मते // 1।18 //  
With a mind free from impurities.
### 1.19
चरतः // 1।19 //  
Engaging in such practice.
### 1.20
ततो ऽस्य योगः प्रवर्तते // 1।20 //  
Then his Yoga begins to function.
### 1.21
दूर-दर्शन-श्रवण-मनन-विज्ञानानि चास्य प्रवर्तन्ते // 1।21 //  
Then develop in him the powers of seeing afar, hearing from a distance, deep reflection, and special knowledge.
### 1.22
सर्व-ज्ञता // 1।22 //  
Becoming omniscient.
### 1.23
मनो-जवित्वम् // 1।23 //  
Attaining the speed of the mind.
### 1.24
काम-रूपित्वम् // 1।24 //  
Taking any desired form.
### 1.25
विकरणः // 1।25 //  
Transcending transformation.
### 1.26
धर्मित्वं च // 1।26 //  
Attaining perfect righteousness.
### 1.27
सर्वे चास्य वश्या भवन्ति // 1।27 //  
All become subject to his will.
### 1.28
सर्वेषां चावश्यो भवति // 1।28 //  
Yet, he is under the will of none.
### 1.29
सर्वांश्चाविशति // 1।29 //  
He pervades all beings.
### 1.30
सर्वेषां चानावेश्यो भवति // 1।30 //  
Yet, none pervades him.
### 1.31
सर्वे चास्य वध्या भवन्ति // 1।31 //  
All become powerless against him.
### 1.32
सर्वेषां चावध्यो भवति // 1।32 //  
Yet, he is invincible against all.
### 1.33
अभीतः // 1।33 //  
Fearless.
### 1.34
अक्षयः // 1।34 //  
Imperishable.
### 1.35
अजरः // 1।35 //  
Unaging.
### 1.36
अमरः // 1।36 //  
Immortal.
### 1.37
सर्वत्र चाप्रतिहतगतिर्भवति // 1।37 //  
He moves everywhere without obstruction.
### 1.38
इत्येतैर्गुण-युक्तो [गुणैर्युक्तो] भगवतो महादेवस्य महागणपतिर्भवति // 1।38 //  
Thus, endowed with these qualities, he becomes a great leader of the Gaṇas of Bhagavān Mahādeva.
### 1.39
अत्रेदं ब्रह्म जपेत् // 1।39 //  
Here, one should chant this Brahman mantra.
### 1.40
[[TA#10.17]]
सद्यो-जातं प्रपद्यामि // 1।40 //  
I surrender to the one born instantly.
### 1.41
सद्यो-जाताय वै नमः // 1।41 //  
Salutations to the instant-born one.
### 1.42
भवे भवे नातिभवे // 1।42 //  
In every birth, never beyond existence.
### 1.43
भजस्व माम् // 1।43 //  
Grant me your devotion.
### 1.44
भवोद्भवः // 1।44 //  
He is the source of existence.

The left. // 2। 1 //

देवस्य // 2। 2 //  
Of the deity. // 2। 2 //

ज्येष्ठस्य // 2। 3 //

रुद्रस्य // 2। 4 //  
Of Rudra. // 2। 4 //

Furnished seat . // 2। 5 //

And the austerity that is intensely practiced. // 2। 16 //  
### 2.17

### 2.20

No other devotion is like that to Śankara. // 2। 20 //  
### 2.21
अत्रेदं ब्रह्म जपेत् // 2। 21 //  
Here, one should chant the Brahman. // 2। 21 //

# c3
अथ  तृतीयो ऽध्यायः    

### 3.1
अव्यक्त-लिङ्गी // 3।1 //  
One who has an unmanifested Liṅga (identity or form).
### 3.2
व्यक्ताचारः // 3।2 //  
Yet, externally follows the common conduct.
### 3.3
अवमतः // 3।3 //  
He is disregarded.
### 3.4
सर्व-भूतेषु // 3।4 //  
Among all beings.
### 3.5
परिभूयमानश्चरेत् // 3।5 //  
He should move about being reviled (without reacting).
### 3.6
अपहत-पाप्मा // 3।6 //  
He is devoid of sin.
### 3.7
परेषां परिवादात् // 3।7 //  
By the slander of others.
### 3.8
पापं च तेभ्यो ददाति // 3।8 //  
He transfers sin to them.
### 3.9
सुकृतं च तेषामादत्ते // 3।9 //  
And takes their merits.
### 3.10
तस्मात् // 3।10 //  
Therefore...
### 3.11
प्रेतवच्चरेत् // 3।11 //  
He should behave like a ghost (remaining unnoticed).
### 3.12
क्राथेत वा // 3।12 //  
He may make noise.
### 3.13
स्पन्देत वा // 3।13 //  
Or tremble.
### 3.14
मण्टेत वा // 3।14 //  
Or wander aimlessly.
### 3.15
शृङ्गारेत वा // 3।15 //  
Or indulge in amorous gestures.
### 3.16
अपि तत्कुर्यात् // 3।16 //  
He may do even that.
### 3.17
अपि तद्भाषेत् // 3।17 //  
He may even speak in that manner.
### 3.18
येन परिभवं गच्छेत् // 3।18 //  
By which he becomes despised.
### 3.19
परिभूयमानो हि विद्वान् कृत्स्न-तपा भवति // 3।19 //  
For the wise one, by being insulted, attains complete ascetic purification.
### 3.20
अत्रेदं ब्रह्म जपेत् // 3।20 //  
Here, one should chant this Brahman mantra.
### 3.21
[[TA#10.19]]
अघोरेभ्यः // 3।21 //  
From the Aghoras (fierce forms of Rudra)...
### 3.22
अथ घोरेभ्यः // 3।22 //  
And also from the Ghoras (more intense forms)...
### 3.23
घोर-घोरतरेभ्यश्च // 3।23 //  
And from those even more terrible than the terrible.
### 3.24
सर्वेभ्यः // 3।24 //  
From all of them...
### 3.25
शर्व-सर्वेभ्यः // 3।25 //  
From Śarva (Rudra) and all (his forms)...
### 3.26
नमस्ते अस्तु रुद्र-रूपेभ्यः // 3।26 //  
Salutations be to all the forms of Rudra.

# c4
अथ  चतुर्थो ऽध्यायः    
### 4.1
गूढ-विद्या तप आनन्त्याय प्रकाशते // 4। 1 //  
The secret knowledge shines forth for the sake of eternal austerity. // 4। 1 //  
### 4.2
गूढ-व्रतः // 4। 2 //  
One whose vow is secret. // 4। 2 //  
### 4.3
गूढ-पवित्र-वाणिः // 4। 3 //  
One whose speech is secretly pure. // 4। 3 //  
### 4.4
सर्वाणि द्वाराणि पिधाय // 4। 4 //  
Having closed all doors. // 4। 4 //  
### 4.5
बुद्ध्या // 4। 5 //  
With intellect. // 4। 5 //  
### 4.6
उन्मतवदेको विचरेत लोके // 4। 6 //  
One should wander alone in the world like a madman. // 4। 6 //  
### 4.7
कृतान्नमुत्सृष्टमुपाददीत // 4। 7 //  
One should accept food that is offered or discarded. // 4। 7 //  
### 4.8
उन्मत्तो मूढ इत्येवं मन्यन्ते इतरे जनाः // 4। 8 //  
Other people consider him mad and foolish. // 4। 8 //  
### 4.9
असम्मानो हि यन्त्राणां सर्वेषामुत्तमः स्मृतः // 4। 9 //  
For he is remembered as the best among all instruments, though without recognition. // 4। 9 //  
### 4.10
इन्द्रो वा अग्रे असुरेषु पाशुपतमचरत् // 4। 10 //  
Indeed, Indra first practiced Pāśupata among the Asuras. // 4। 10 //  
### 4.11
स तेषामिष्टापूर्तमादत्त // 4। 11 //  
He took away their sacrifices and meritorious deeds. // 4। 11 //  
### 4.12
मायया सुकृतया समविन्दत // 4। 12 //  
Through his well-performed illusion, he conquered them. // 4। 12 //  
### 4.13
निन्दा ह्येषानिन्दा तस्मात् // 4। 13 //  
For this censure is indeed praise. // 4। 13 //  
### 4.14
निन्द्यमानश्चरेत् // 4। 14 //  
One should behave while being censured. // 4। 14 //  
### 4.15
अनिन्दित-कर्मा // 4। 15 //  
One whose actions are beyond censure. // 4। 15 //  
### 4.16
सर्व-विशिष्टो ऽयं पन्थाः // 4। 16 //  
This is the most distinguished path. // 4। 16 //  
### 4.17
सत्पथः // 4। 17 //  
The path of the righteous. // 4। 17 //  
### 4.18
कुपथास्त्वन्ये // 4। 18 //  
But others are on wrong paths. // 4। 18 //  
### 4.19
अनेन विधिना रुद्र-समीपं गत्वा // 4। 19 //  
Having gone near Rudra by this method. // 4। 19 //  
### 4.20
न कश्चिद्ब्राह्मणः पुनरावर्तते // 4। 20 //  
No one who has attained Brahman returns again. // 4। 20 //  
### 4.21
अत्रेदं ब्रह्म जपेत् // 4। 21 //  
Here, one should chant the Brahman. // 4। 21 //  
### 4.22
[[TA#10.20]]
तत्पुरुषाय विद्महे // 4। 22 //  
We meditate upon that Supreme Being. // 4। 22 //  
### 4.23
महा-देवाय धीमहि // 4। 23 //  
We contemplate upon the Great God. // 4। 23 //  
### 4.24
तन्नो रुद्रः प्रचोदयात् // 4। 24 //  
May Rudra inspire us. // 4। 24 //

# c5
अथ  पञ्चमो ऽध्यायः    
### 5.1
असङ्गः // 5।1 //  
Unattached.
### 5.2
योगी // 5।2 //  
A yogi.
### 5.3
नित्यात्मा // 5।3 //  
Eternal in self.
### 5.4
अजः // 5।4 //  
Unborn.
### 5.5
मैत्रः // 5।5 //  
Friendly (compassionate).
### 5.6
अभिजायते // 5।6 //  
He is born anew (spiritually realized).
### 5.7
इन्द्रियाणामभिजयात् // 5।7 //  
By conquering the senses.
### 5.8
रुद्रः प्रोवाच तावत् // 5।8 //  
Rudra thus declared.
### 5.9
शून्यागार-गुहा-वासी // 5।9 //  
One who dwells in empty places or caves.
### 5.10
देव-नित्यः // 5।10 //  
Ever-divine.
### 5.11
जितेन्द्रियः // 5।11 //  
Master of the senses.
### 5.12
षण्मासान्नित्य-युक्तस्य // 5।12 //  
For six months, if he remains ever absorbed (in yoga).
### 5.13
भूयिष्ठं संप्रवर्तते // 5।13 //  
He advances greatly (in realization).
### 5.14
भैक्ष्यम् // 5।14 //  
Alms (as food).
### 5.15
पात्रागतम् // 5।15 //  
Whatever comes into the bowl.
### 5.16
मांसमदुष्यं लवणेन वा // 5।16 //  
Meat unspoilable by salt.
### 5.17
आपो वापि यथा-कालमश्नीयादनुपूर्वशः // 5।17 //  
Or just water, consuming in due time and in proper order.
### 5.18
गो-धर्मा मृग-धर्मा वा // 5।18 //  
Following the way of cows (gentleness) or the way of deer (detachment).
### 5.19
अद्भिरेव शुचिर्भवेत् // 5।19 //  
He remains pure by water alone.
### 5.20
सिद्ध-योगी न लिप्यते कर्मणा पातकेन वा // 5।20 //  
The perfected yogi is not tainted by actions or sins.
### 5.21
ऋचमिष्टामधीयीत गायत्रीमात्मयन्त्रितः // 5।21 //  
He should recite the desired Vedic hymn (ṛc), the Gāyatrī, with self-control.
### 5.22
रौद्रीं वा बहु-रूपीं वा // 5।22 //  
Or the fierce (Raudrī) or the many-formed (mantra).
### 5.23
अतो योग प्रवर्तते // 5।23 //  
From this, yoga is initiated.
### 5.24
ओं-कारमभिध्यायीत // 5।24 //  
He should meditate on Om.
### 5.25
हृदि कुर्वीत धाराम् // 5।25 //  
He should establish its stream in the heart.
### 5.26
ऋषिर्विप्रो महानेषः // 5।26 //  
This sage is a great seer and knower (vipra).
### 5.27
वाग्विशुद्धः // 5।27 //  
Purified in speech.
### 5.28
महेश्वरः // 5।28 //  
He is the great lord (Maheshvara).

श्मशान-वासी // 5।30 //  
One who dwells in cremation grounds.
### 5.31
धर्मात्मा // 5।31 //  
Righteous in soul.
### 5.32
यथा-लब्धोपजीवकः // 5।32 //  
Living on whatever is obtained.
### 5.33
लभते रुद्र-सायुज्यम् // 5।33 //

सदा रुद्रमनुस्मरेत् // 5।34 //

बुद्ध्या // 5।36 //  
By wisdom.
### 5.37
सञ्चित्तम् // 5।37 //

स्थापयित्वा च रुद्रे // 5।38 //

### 5.39
एकः क्षेमी सन् वीत-शोकः // 5।39 //

### 5.40
अप्रमादी गच्छेद्दुःखानामन्तमीशप्रसादात् // 5।40 //  
With diligence, he reaches the end of suffering by the grace of the Lord.
### 5.41
अत्रेदं ब्रह्म जपेत् // 5।41 //  
Here, one should chant this Brahman mantra.
### 5.42
[[TA#10.21]]
ईशानः सर्व-विद्यानाम् // 5।42 //  
Īśāna (the supreme Lord) of all knowledge.
### 5.43
ईश्वरः सर्व-भूतानाम् // 5।43 //  
The Lord of all beings.
### 5.44
ब्रह्मणो ऽधिपतिर्ब्रह्मा // 5।44 //  
The master of Brahman, the creator.
### 5.45
शिवो मे अस्तु // 5।45 //  
May Śiva be with me.
### 5.46
सदा // 5।46 //  
Always.
### 5.47
शिवः // 5।47 //  
Śiva.
